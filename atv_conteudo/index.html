O que é Lógica de Programação?

Lógica de programação é a técnica de encadear pensamentos para atingir determinado objetivo. É o passo fundamental para qualquer pessoa que deseja se tornar um desenvolvedor de software.

Como se fosse a forma de organizar o pensamento para resolver um problema usando passos claros, como se fosse uma receita. É pensar antes de programar. Por exemplo, se eu quiser criar um programa para ver se alguém passou na prova, eu preciso: 1. Guardar a nota em uma variável. 2.Usar um if para verificar se a nota é maior ou igual a 7. 3.Mostrar “Aprovado” ou “Reprovado”.

DICA: "Entendendo Algoritmos"

Este livro é uma leitura essencial. Ele explica conceitos complexos, como busca binária, recursão e grafos, de uma forma visual e muito simples (com ilustrações divertidas de ratinhos e outros personagens).

Conteúdos Principais:

  • Algoritmos: Sequência de passos para resolver um problema.
  • Variáveis: Espaços na memória para guardar dados.
  • Estruturas de Repetição: Loops para automatizar tarefas (for, while).
  • Estruturas de Dados: Formas de organizar informações (listas, arrays, pilhas).
